# Kaffeemachine für BlueJ

## Features
* Touch Funktion
* Kaffees
  * Latte Macchiato
  * Kaffee
  * Cappuccino
  * Heiße Milch
* Wasserfüllstand
* Reinigungsfunktion am Ende (anderer Becher)

## How to use
#### Kaffeetasse wechseln
Einfach 3x auf die Kaffeetasse klicken

### Wasser auffüllen
Einfach 3x auf den Wassertank klicken
